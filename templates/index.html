Katvan

A simple, cross-platform editor for Typst files with a bias for Right-to-Left languages

Download View on GitHub

Focus on Right-to-Left

Write (and edit!) complex technical documents in languages using bi-directional scripts without losing your mind:

  • Automatic isolation of math and code blocks
  • Smart line directionality heuristics
  • Accurate cursor movements
  • Easy-to-access manual BiDi controls

Typst Native

Comes fully-integrated out of the box with the Typst compiler:

  • Intelligent syntax highlighting and spell checking
  • Live preview with editor cursor synchronization
  • Auto-completion
  • Quick access to Typst documentation
  • Syntax and context aware editing aides
  • And more!

Cross-Platform and Local

Runs completely locally as a native application across Linux, Windows, and macOS. Quick startup, responsive editing experience, and you are in full control of your data.
